# PowerElectronicsLibrary

A Modelica library for thermal modelling of power electronic components and PCB heat transfer.

## Overview

PowerElectronicsLibrary is an open-source Modelica library being developed to model the thermal behaviour of power electronic systems. The primary focus is on PCB heat transfer, component-level thermal modelling, and future support for convection, radiation, and cooling systems.

The library is currently under active development.
